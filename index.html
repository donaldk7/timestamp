Timestamp Microservice

User Story

  • Check to see whether the parameter string contains either a unix timestamp or a natural language date (example: January 1, 2016)
  • Returns both the Unix timestamp and the natural language form of that date
  • If it does not contain a date or Unix timestamp, it returns null for those properties

Example Usage

https://timestamp132.herokuapp.com/November 30, 2016

https://timestamp132.herokuapp.com/1234567890

Example Output

{"unix":"1234567890","natural":"February 13, 2009"}
